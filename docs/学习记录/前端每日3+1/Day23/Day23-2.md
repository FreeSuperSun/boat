---
title: 说说CSS的优先级是如何计算的
---

CSS 根据选择符来计算优先级,采用分级计算的方式,上一级永远大于下一级,同级之间求和.

| 选择符             | 优先级 | 备注 |
| ------------------ | ------ | ---- |
| !important         | 无穷大 |
| 内联               | 1000   |
| id 选择符          | 100    |
| 类选择符 属性 伪类 | 10     |
| 元素选择符 伪元素  | 1      |
| 通用选择符         | 0      |
| 继承               | -1     |
