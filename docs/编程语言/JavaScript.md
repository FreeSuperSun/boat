# JavaScript

## 字符串 String


#### String.prototype.charCodeAt(index)

返回字符串指定位置的UTF-16数字.

#### String.prototype.match(regexp)

正则匹配

#### String.prototype.slice(beginIndex,endIndex)

返回子字符串,不修改原字符串.

#### String.prototype.substring(beginIndex,endIndex)

slice的退化版.索引不能为负数.
