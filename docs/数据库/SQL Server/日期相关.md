# 日期相关

## 相关函数

#### `DATEADD` 日期增减

```sql
DATEADD(datepart,number,date)
//datepart 要增减的日期部分 如年月日等
//number 增减的数值,正数为增加,负数为减少
//date 要进行增减的日期
//返回增减后的日期,datetime类型
``` 

#### `DATEDIFF` 日期比较

```sql
DATEDIFF(datepart,startdate,enddate)
//datepart 要比较的日期部分 如年月日等
//startdate enddate 被比较的日期,为enddate-startdate
```
