# 杂项

存放一些暂时没有整理分类的内容.

#### Github短信验证码没有中国

手动给下拉框加上中国就可以

```js
var option = new Option("China +86","+86");
option.selected = true;
document.getElementById('countrycode').options.add(option, 0);
```
[Github短信验证码没有中国区的解决办法](https://blog.csdn.net/aa464971/article/details/83860319)
